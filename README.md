## Stack
- **Next.js** — App Router, with Next Actions as the server layer for data fetching
- **TanStack React Query** — manages fetching, caching, and pre-fetching the next level during the animation
- **motion/react** (formerly Framer Motion) — `useAnimate` drives the fragment flight, `motion` components handle bar entrances and UI transitions
